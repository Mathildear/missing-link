A font created by

Missing Link is a font with a soft and elegant look. It's distinctive features are the gap in every letter making it forever incomplete. The font is working in several boldness making it able to be customized by whoever using it to fit their design. The inspiration for the design of the font is mostly line art, as I wanted the softness og simple from this art style.
